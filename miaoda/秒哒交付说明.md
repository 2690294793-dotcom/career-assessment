# 秒哒交付说明 — 职趣星图（自在生涯工作室）

## 一、项目概况

这是一个**职业规划咨询预约系统**，前端是纯静态页面（HTML+CSS+JS），用 GitHub Pages 部署。

**现在需要把数据后端从 Supabase 迁移到秒哒。**

### 2. 提供 REST API

所有数据操作集中在 `js/config.js` 中，通过 `miaodaRequest(method, path, body)` 函数调用。

**API 调用格式：**

```
方法：GET / POST / PATCH / DELETE
鉴权：Bearer Token（Header: Authorization）
请求体：JSON
响应体：JSON
```

**接口清单（23 个）：**

| 方法 | 路径 | 说明 |
|------|------|------|
| POST | /submissions | 提交测评数据 |
| GET | /submissions | 获取所有测评 |
| DELETE | /submissions/:id | 删除测评 |
| POST | /report_exports | 提交报告导出 |
| GET | /report_exports | 获取所有报告 |
| POST | /appointments | 创建预约 |
| GET | /appointments | 获取预约（支持 ?status=&mentor_id=&student_name= 过滤） |
| PATCH | /appointments/:id | 更新预约状态 |
| PATCH | /appointments/by-no/:no | 保存学生反馈 |
| DELETE | /appointments/:id | 删除预约 |
| GET | /appointments/stats | 获取预约统计 |
| GET | /mentor_schedule | 获取时间表（支持 ?mentor_id= 过滤） |
| POST | /mentor_schedule | 添加时间段 |
| DELETE | /mentor_schedule/:id | 删除时间段 |
| GET | /mentor_settings | 获取导师设置（支持 ?mentor_id= 过滤） |
| POST | /mentor_settings | 创建导师设置 |
| PATCH | /mentor_settings/:mentor_id | 更新导师设置 |
| GET | /articles | 获取文章列表 |
