四、前端整体项目架构

纯前端多页面应用，无构建工具，无框架依赖，所有页面为独立 HTML 文件，内联 CSS 和 JavaScript。数据层采用 Supabase 云数据库 + localStorage 本地存储双轨制，Supabase 不可用时自动降级到 localStorage。

三层架构：
- 页面层：根目录下所有 HTML 文件，页面间通过 window.location.href 跳转，URL 参数传递数据。
- 逻辑层：js 目录下的 JavaScript 文件。
- 资源层：css 目录提供全局样式，img 目录提供图片资源。

关键文件：
- js/config.js：Supabase 配置与全部数据库操作函数（saveSubmission、getAppointments、createAppointment、saveConsultationNotes、getMentorSchedule、addScheduleSlot、deleteScheduleSlot、getMentorSettings、updateMentorSettings、saveFeedback 等）
- js/articles-data.js：26 篇预设公众号文章，Supabase 兜底数据
- js/student.js：霍兰德职业兴趣类型常量 CAREER_TYPES
- js/teacher.js：教师端常量（TEACHER_PASSWORD、TEST_INFO）
- database.sql：建表脚本与 RLS 策略
- push-files.js：GitHub Contents API 批量推送脚本

五、所有页面文件功能说明

index.html
首页，SVG 六边形导航，登录拦截 requireLogin()，学生/教师端切换 toggle，手机框架设计 390x844px。

login.html
微信快捷登录风格，首次登录填写姓名和学号，学校固定为广州职业技术大学且只读。doQuickLogin()/doSimpleLogin()。

profile.html
完善个人资料（基本信息、教育经历、工作经历、项目经历），数据存 localStorage 的 resumeData 键。

resume.html
我的页面，顶部头像和姓名，下方菜单：我的预约、我的测评、我的规划方案。预约状态为 completed 时显示「提交反馈」或「查看报告」按钮。

about.html
关于我们，平台简介与导师团队展示。

assessment.html
四种测评：霍兰德职业兴趣测评（60题，RIASEC每维度10题）、MBTI性格测评（60题，EI/SN/TF/JP每维度15题，24道反转题）、大五人格测评（OCEAN）、DISC行为风格测评。引入 Chart.js 绘制雷达图，html2canvas 支持报告导出。支持学校验证码解锁完整报告。
注意：霍兰德和MBTI题目已于2026-07-07更新为60题版本，来源O*NET和OEJTS开源量表，但assessment.html本地修改完成尚未推送GitHub。

consult-flow.html
预约咨询流程：选导师→选类型→选日期时间→填写确认信息→提交→反馈评价→报告。支持URL参数?apt=APTxxx直接进入反馈或报告步骤。日期时间与导师时间管理同步，只显示导师开放时段的日期。

articles.html
资讯中心，Supabase优先加载、预设数据兜底。支持搜索（关键词高亮）、分类过滤、文章详情弹窗。文章来源标识：微信原文(绿色badge)/内容摘要(橙色badge)。跳转原文按钮。

goals.html
目标进度管理，子步骤、进度条、折叠展开，数据存localStorage的career_goals_v2键。

teacher.html
教师端管理后台，桌面布局 max-width 1200px，密码登录。四个标签页：
- 测评管理（查看/筛选/导出CSV）
- 预约管理（审批/咨询记录/删除）
- 咨询记录
- 时间管理（预约开关booking_open + 日期选择器添加时间段 + 列表删除）
注意：添加时间段函数名为handleAddScheduleSlot()，不要用addScheduleSlot()（会和config.js同名函数冲突导致无限递归）。

resume-templates.html
职业规划方案生成，目标职业方向下拉框已更新为广州职业技术大学20个专业对应方向。

setup-guide.html
Supabase部署图文教程。

database.sql
建表脚本：submissions、appointments、mentor_schedule、mentor_settings、articles。含RLS策略、索引、触发器。

七、统一代码编写规范

HTML：每个文件独立页面，DOCTYPE html，html lang="zh-CN"，内联style和script。页面标题格式：页面名 · 自在生涯工作室。移动端viewport加maximum-scale=1.0, user-scalable=no。

CSS：内联style标签，不使用外部CSS文件。:root定义CSS变量（--primary、--bg、--card等）。手机框架用.phone类390x844px。字体：-apple-system, BlinkMacSystemFont, 'PingFang SC', 'Microsoft YaHei', sans-serif。按钮:active { transform: scale(0.97); }。

JavaScript：内联在页面body末尾。函数声明用function关键字。异步用async/await。XSS防护用esc()。提示用showToast()。localStorage键名：app_user_info、resumeData、career_goals_v2、assessment_results。学校固定广州职业技术大学。数据库操作通过js/config.js函数调用。新增功能内联在对应页面，不创建新外部JS。
